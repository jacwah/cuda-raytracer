# CUDA ray tracer

Project for course DD2360 Applied GPU Programming.

```
Makefile            Build code
build_tegner.sh     Build all configurations for Tegner
fig/                Generate figures
makedat.sh          Post-process timing data
perf_tegner.sh      Job script for GPU timing
perf_tegner_cpu.sh  Job script for CPU timing
ray0.cu             Implementation, fwrite
ray1.cu             Implementation, mmap
ray2.cu             Implementation, streams
raytracing.py       Reference CPU implementation
report/             Project report
```
